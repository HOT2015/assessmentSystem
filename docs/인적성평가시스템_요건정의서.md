# 인적성 평가시스템 요건정의서

## 1. 프로젝트 개요

### 1.1 프로젝트명
**통합 온라인 인적성 평가시스템**

### 1.2 프로젝트 목적
- 기술 역량 + 인성 + 문제 해결력을 통합적으로 평가하는 온라인 시스템 구현
- 자동 채점과 리포트 생성으로 평가 프로세스 전체 자동화
- 신입/경력 개발자 대상 맞춤형 평가 제공

### 1.3 주요 가치
- 객관적이고 일관된 평가 기준 제공
- 시간과 비용 효율적인 평가 프로세스
- 데이터 기반의 의사결정 지원

---

## 2. 시스템 요구사항

### 2.1 기능적 요구사항 (Functional Requirements)

#### 2.1.1 사용자 관리
- **지원자 등록/로그인**
  - 기본 정보 입력 (이름, 이메일, 경력, 지원 직무)
  - 지원자 고유 ID 생성
  - 평가 이력 관리

- **관리자 기능**
  - 지원자 목록 조회
  - 평가 결과 통계 확인
  - 시스템 설정 관리

#### 2.1.2 평가 시스템
- **기술 지식 평가**
  - 객관식 문제 (Java, DB, 네트워크, 알고리즘 등)
  - 주관식 문제 (코딩 테스트, 설계 문제)
  - 난이도별 문제 분류 (초급/중급/고급)

- **인성 검사**
  - 협업 능력 평가
  - 책임감 및 성장 마인드셋
  - 리더십 및 커뮤니케이션 스타일
  - 스트레스 대응 능력

- **문제 해결력 평가**
  - 시나리오 기반 문제
  - 실무 상황 시뮬레이션
  - 창의적 사고력 측정

#### 2.1.3 자동 채점 시스템
- **객관식 자동 채점**
  - 정답 비교 기반 즉시 채점
  - 부분 점수 부여 (선택지별 가중치)

- **주관식 자동 채점**
  - 키워드 매칭 기반 채점
  - 코드 실행 및 결과 검증
  - 정확도 및 완성도 평가

- **인성 검사 분석**
  - 응답 패턴 분석
  - 성향별 점수 계산
  - 일관성 검증

#### 2.1.4 리포트 생성
- **개인별 상세 리포트**
  - 영역별 점수 및 백분율
  - 강점 및 개선점 분석
  - 상위/하위 영역 표시
  - 맞춤형 피드백 제공

- **비교 분석 리포트**
  - 동일 직무 지원자 간 비교
  - 평균 점수 대비 위치
  - 상대적 순위 표시

### 2.2 비기능적 요구사항 (Non-Functional Requirements)

#### 2.2.1 성능 요구사항
- 동시 접속자 100명 지원
- 페이지 로딩 시간 3초 이내
- 평가 완료 시간 30분 이내

#### 2.2.2 보안 요구사항
- 개인정보 암호화 저장
- 세션 관리 및 타임아웃
- 접근 권한 제어

#### 2.2.3 사용성 요구사항
- 직관적인 사용자 인터페이스
- 반응형 웹 디자인
- 접근성 고려 (WCAG 2.1 준수)

## 3. 시스템 아키텍처

### 3.1 기술 스택
- **Backend**: Python (Flask/Django)
- **Frontend**: HTML, CSS, JavaScript
- **Database**: SQLite (개발) / PostgreSQL (운영)
- **Data Format**: JSON

### 3.2 시스템 구조
```
인적성평가시스템/
├── app/
│   ├── __init__.py
│   ├── models/          # 데이터 모델
│   ├── views/           # 뷰 함수
│   ├── templates/       # HTML 템플릿
│   ├── static/          # CSS, JS, 이미지
│   └── utils/           # 유틸리티 함수
├── data/
│   ├── questions.json   # 문제 데이터
│   └── results.json     # 결과 데이터
├── tests/               # 테스트 코드
├── requirements.txt     # 의존성
└── README.md           # 프로젝트 설명
```

  "position": "지원직무",

  "category": "카테고리(기술/인성/문제해결)",

## 5. 평가 프로세스

### 5.1 평가 흐름
1. **지원자 등록** → 기본 정보 입력
2. **평가 시작** → 문제 유형별 순차 진행
3. **기술 평가** → 객관식 + 주관식 문제
4. **인성 검사** → 성향 파악 질문
5. **문제해결력 평가** → 시나리오 기반 문제
6. **자동 채점** → 즉시 결과 계산
7. **리포트 생성** → 상세 분석 제공

### 5.2 평가 시간 배분
- **기술 평가**: 15분 (객관식 10분 + 주관식 5분)
- **인성 검사**: 10분
- **문제해결력 평가**: 5분
- **총 평가 시간**: 30분
